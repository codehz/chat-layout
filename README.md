# chat-layout

To install dependencies:

```bash
bun install
```

To type-check:

```bash
bun run typecheck
```

To build distributable files:

```bash
bun run dist
```

To build example:

```bash
bun run example
```

This project was created using `bun init` in bun v1.1.37. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
